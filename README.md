# Dynamic Task Display & Modal View

## Overview

In this project, tasks from the **given initial data file** dynamically display on the DOM using JavaScript . Tasks are placed into the correct Kanban board columns based on their status, and clicking a task opens a modal where users can view and modify task details. The project emphasizes DOM manipulation, event handling, modular JavaScript structure, and responsive UI implementation.

## Technologies Used

Language: JavaScript ,HTML & CSS
Environment: Web Browser
